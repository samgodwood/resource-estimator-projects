# RE_intro

## Usage

To run the notebooks:
1. Activate the `resource-estimator-env` environment:
    ```bash
    conda activate resource-estimator-env
    ```
2. Open the Jupyter notebook interface:
    ```bash
    jupyter notebook
    ```
3. Navigate to *RE_intro* and open the notebooks to begin.
